# School Management System 🎓

A comprehensive Full-Stack application designed to streamline school administration, student registration, and academic tracking. Built with a robust **.NET Core Web API** and a dynamic **Angular** frontend.

## 🏗️ Project Structure

The repository is organized into two main modules:

* **/Backend**: ASP.NET Core Web API, Entity Framework Core, and SQL Server.
* **/Frontend**: Angular 17+, Bootstrap, and RxJS.
* **/SQL**: Database schema scripts and stored procedures.

---

## 🚀 Features

- **Student Management**: Automated Student ID generation and registration.
- **Security**: JWT (JSON Web Token) Authentication with Angular AuthGuards.
- **Architecture**: Repository Pattern with Dependency Injection.
- **UI/UX**: Responsive dashboard built with Angular Material and Bootstrap.

---

## 🛠️ Tech Stack

**Backend:**
- .NET Core 8.0
- Entity Framework Core (Code First/DB First)
- SQL Server
- JWT Authentication

**Frontend:**
- Angular
- TypeScript
- HTML5 & SCSS
- Bootstrap 5
